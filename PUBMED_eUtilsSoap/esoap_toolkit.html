Using NCBI Entrez Utilities Web Service with Microsoft SOAP Toolkit 3.0

Updated: December 14, 2010

 

  • Creating an NCBI Web Service Client Project
  • SOAP E-Utilities API


SOAP Interface for E-Utilities was tested with:

  • Microsoft Visual Basic 6.5 in Microsoft Office Word 2007
  • Microsoft SOAP Toolkit 3.0

Creating an NCBI Web Service Client Project

Walkthrough: Accessing the NCBI Entrez Utilities Web service from Word macros.

To create Windows application:

  1. In Word open Visual Basic Eitor (Alt+F11)
  2. Add Soap Library reference to project: select Tools->References... menu, check Microsoft Soap Type Library 3.0 and click OK.
  3. Add form to project: select Insert->UserForm menu
  4. From the Toolbox, drag a Button to the form
  5. Double-click the button on form to create an event-handling method for this button.
  6. In the CommandButton1_Click method enter the following code: 
    
    ' eInfo utility returns a list of available databases        
    Dim soapclient As MSSOAPLib30.SoapClient30
    On Error GoTo FAIL
    Set soapclient = New 
    SoapClient30    soapclient.MSSoapInit "http://eutils.ncbi.nlm.nih.gov/entrez/eutils/soap/v2.0/eutils_MS.wsdl"
    
    ' call NCBI EInfo utility
    Dim X1
    Set X1 = soapclient.run_eInfo_MS("", "", "")
    MsgBox X1.Context.XML
    Exit Sub
    
    FAIL:
        MsgBox soapclient.Detail
        
  1. Run application.
